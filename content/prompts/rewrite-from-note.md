# プロンプト: 過去の note 記事をリライトする

過去に書いた note の記事を、このメディアの文体・コンテンツループに沿った形でリライトするためのプロンプト。

---

## ユーザープロンプト（テンプレート）

```
以下は過去に note で書いた記事です。これを「地域課題、どう解く？」用にリライトしてください。

【セクション】
{analysis | prototype | experiments | logs}

【コンテンツループ上の位置づけ】
事実 / 分析 / 仮説・プロト / 検証 のどれか

【リライトの方針】
- 文体方針（system prompt 参照）に従う
- 元記事の体験・事実は残す
- プロダクト思考の構造化を加える（症状と課題を分ける、フレームワークで分解する など）
- 「次のステップ」をコンテンツループの次ステージに接続する一段落を入れる

【元記事 URL】
{URL}

【元記事本文】
（必要なら貼る）

---

出力フォーマット:
- 冒頭に MDX frontmatter
- 本文は Markdown（h2 区切り）
```
