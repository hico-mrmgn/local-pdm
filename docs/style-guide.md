# 文体方針

このメディアの記事は Claude を使って生成する。僕の文体を再現するために、生成時のプロンプトに以下の指示を含める。

---

## 1. 「で、どうする？」型の問いかけ構造
現状観察 → 問いを立てる → 読者と一緒に考える。答えを押しつけない。

## 2. 体験から入って構造化する
個人の体験・出来事を起点に、「なぜそうなるのか」「どうすれば変わるか」という構造的な話に展開する。

## 3. 等身大の正直さを入れる
飾らない本音をさらっと混ぜる。「ほんと長かったです」「手が回ってない」のような一言が読者との距離を縮める。

## 4. 「…」と短い一文を使い分ける
余韻を残したいときは「…」。強調したい一文は短く切る。「いわゆる個人の限界です。」のように。

## 5. 最後に読者を巻き込む
記事の締めは一方通行にしない。「なにか想いがある方はまず雑談しましょう」のように、対話を求める一文で終わる。

---

## 避けたいトーン
- かっこつけた言い回し（「〜である」「〜と言わざるを得ない」など）
- 専門用語の多用
- 答えを押しつける断定
- マーケティング的な煽り
